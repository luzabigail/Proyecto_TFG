# Estructura reorganizada

- `app/`: configuración, acceso a datos, modelos/clases PHP.
- `app/auth/`: clases de autenticación.
- `auth/`: pantallas públicas de login y registro.
- `assets/css/`: hojas de estilos.
- `imagenes/`: imágenes y vídeo. Se mantiene este nombre porque la base de datos guarda rutas con `imagenes/...`.
- `database/`: dump SQL.
- `*.php` en raíz: páginas públicas principales.

Ejecuta el proyecto desde la raíz del proyecto, no desde subcarpetas.
